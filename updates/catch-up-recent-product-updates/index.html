Catch-up: recent product updates

A number of these updates have been sitting in my personal notes with every intention of sharing them sooner. Since AirShop Latest is just getting started, this first catch-up brings the recent work into one place.

Recent work across AirShop includes faster quoting, better inventory workflows, cleaner printing, stronger imports, and more useful navigation throughout the app. More regular updates from here on out.

- Justin

Faster quoting and sales workflows

  • The quote builder has been significantly optimized, reducing interruptions while building or revising a quote.
  • Quotes can now be duplicated, making repeat work and similar jobs faster to start without rebuilding from scratch.
  • Inquiries can open in Fusion, and won quotes can trigger a webhook for workflows through Zapier into tools like Xero or QuickBooks.

Inventory improvements

  • Inventory fields now support simple math where it is useful, such as entering 5x2 for quantities, prices, or lead-time style fields.
  • The item page save and discard experience now makes pending changes clearer before they are kept or discarded.
  • Items can now be duplicated to create similar inventory records faster.

Kanban cards, labels, and printing

  • Kanban cards and bin labels now print directly instead of relying on PDF export, keeping output much closer to the web preview.
  • Kanban cards now include an ORDER field, a refined field order with Cold moved below Deliver, and tooltips for mapped inventory fields.
  • Search and filtering were added for Kanban cards, bin labels, and location tags. Location tag printing is also available.

Label readability guidance

  • Best: 0 to 18 characters, for the largest and most readable text.
  • Fits: 19 to 35 characters, still on one line but at a smaller size.
  • Wraps: 36 or more characters, wrapping to two lines.

QR navigation, docs, and imports

  • The QR code scanner can be used anywhere in the app for instant navigation.
  • The new docs site at docs.airshop.work is included in search.
  • Bulk imports are available on the Imports page, with QuickBooks import mapping for contacts and businesses.

Customers, support, insights, and cleanup

  • The new Customer Detail page brings customer information into a more focused view, and support chat is available in the app.
  • Insights are available at /stats.
  • Locations can be merged from the main grid or from the three-dot menu on a Location detail page.
  • Roll-up items were improved around saving, optional selections, and multiple-choice behavior.
