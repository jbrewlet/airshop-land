Latest Updates changelog added

AirShop now has a dedicated place for product release notes, workflow improvements, and smaller quality-of-life updates.

The Latest Updates page gives AirShop product changes a permanent home on the public site. It mirrors the short list shown on the AirShop home dashboard, so users can find recent product work without hunting through support messages, emails, or personal notes.

The dashboard and the public site read the same feed: we publish notes once, they show up in both places. Bigger releases also get a full page with a stable URL you can share or bookmark.
