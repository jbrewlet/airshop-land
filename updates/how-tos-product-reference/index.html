How-tos and product reference

AirShop docs are available at docs.airshop.work, with step-by-step walkthroughs, FAQs, and field-by-field guidance.

The docs site gives AirShop users a written reference while they work. It covers practical workflows, common questions, and field-level guidance for areas like imports and inventory.

Docs are also included in AirShop search, so written help is easier to reach from inside the app instead of living in a separate place users have to remember.

What is included

  • Step-by-step walkthroughs for common AirShop workflows.
  • FAQs and product reference material.
  • Import documentation, including bulk import guidance.
  • Searchable help that connects product guidance back to the app.
