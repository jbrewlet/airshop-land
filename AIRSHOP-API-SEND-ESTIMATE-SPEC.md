# Send Estimate API Spec

The implementation lives in **AirShop** (`/api/leads/send-estimate`), not in this repo.

This repo (`airshop-land`) contains:
- **index.html** — Form that POSTs to `https://airshop.work/api/leads/send-estimate`
- **api/send-estimate.js** — Reference/fallback implementation (template layout, logic)

See **REPO-SEPARATION.md** for what belongs in each repo.
